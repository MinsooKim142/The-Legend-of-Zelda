## Program Controls
- WASD to move the character
- R to reset the game
- Q to quit the game
- Z/N to make Link Attck with his sword
- 1,2,3,4,5,6 to make Link use weapons
- Left/Right click to change rooms sequentially
- Middle click on a door to go into that door directly
